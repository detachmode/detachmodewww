White House Down - Marine One

MODELLING // TEXTURING // SHADING





Modeling was done in Autodesk 3ds Max. Texturing was done in Adobe Photoshop and The Foundary Mari. Displacement map for the dents was done in Pixology ZBrush. Shading and rendering was done in Vray.






Manou the Swift - Characters

CHARACTER DESIGN // MODELING // TEXTURING // SHADING // HAIR AND FUR







I did the character designs of all characters in Manou the Swift in ZBrush. Some of them were based on 2D concepts but some were purely designed inside of ZBrush without any 2D concepts.
When the final design was found, I retopologised the mesh in 3ds Max and reprojected the Polypaint from ZBrush to the new topology. Finally I did the fur/feathers with Ornatix and painted the additional fur textures in Mari.
